# phys-sims-utils v0.3.0 release notes

Date: 2026-02-13

## Highlights

- Added mixed parameter-space contracts with numeric (`bounds`) and categorical/discrete (`choices`) parameters.
- Added deterministic mixed/topology optimization coverage.
- Extended canonical reporting summaries with structure metadata (`structure_keys`, `structure_values`).
- Extended agent experiment templates and repository validators with `structure_fields` metadata.
- Added ADR 0007 for mixed-parameter contract evolution.

## Compatibility

- Existing numeric-only workflows remain supported.
- Sobol and CMA-ES remain numeric-only and now fail fast on categorical parameters with explicit errors.

## Validation

- `python -m pre_commit run -a`
- `python -m mypy src tests`
- `python -m pytest -q -m "not slow"`
