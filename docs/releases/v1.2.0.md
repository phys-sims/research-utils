# phys-sims-utils v1.2.0 release notes

Release date: 2026-02-13

## Highlights

v1.2.0 ships the agent-first roadmap with deterministic, typed workflow surfaces:

- Added workflow contracts for adaptation assistance, optimization strategy advisory,
  and graphics/reporting concierge flows.
- Added shared deterministic agent artifact manifest schema across workflow outputs.
- Added structured workflow error contract for invalid requests/unsupported intents.
- Added deterministic adaptation assistant artifacts and checklist generation.
- Added deterministic strategy advisor with rationale and fallback outputs.
- Added chat-oriented graphics/reporting concierge over canonical result contracts.

## API additions

New exports in `phys_sims_utils.agents`:

- `AdaptationAssistantRequest`, `AdaptationAssistantResponse`
- `StrategyAdvisorRequest`, `StrategyAdvisorResponse`
- `GraphicsIntent`, `GraphicsConciergeRequest`, `GraphicsConciergeResponse`
- `AgentArtifactManifest`, `AgentArtifactRecord`, `AgentWorkflowError`
- `run_adaptation_assistant`, `advise_optimization_strategy`, `run_graphics_concierge`

## Validation

- Added contract/determinism/integration tests in `tests/test_agent_workflows_v04.py`.
- Existing agent tooling tests remain in `tests/test_agents_tools.py`.

## Documentation and architecture

- Added ADR-0008 for v1.2 workflow contract decisions.
- Updated `docs/how-to-use-agents.md` with v1.2 usage guidance.
- Updated `STATUS.md` and roadmap completion state.
